# CNC1 UserPanel Extension

Простое расширение для панели пользователя/администратора сайта CNC1 (Bitrix).

## Что это

Расширение добавляет собственную панель и всплывающее окно для удобной работы на сайте `https://cnc1.ru`.

## Стек

- TypeScript
- Preact
- Vite
- Tailwind CSS
- WXT (для сборки и локальной разработки расширения)

## Запуск локально

1. Установите зависимости:

```bash
pnpm install
```

2. Запустите режим разработки:

```bash
pnpm run dev
```

3. Откройте браузер, перейдите в страницу расширений и подключите папку проекта как распакованное расширение.

### Ручная установка для live-разработки

#### Chrome / Edge

1. Запустите `pnpm run dev`.
2. В Chrome откройте `chrome://extensions` (в Edge — `edge://extensions`).
3. Включите `Режим разработчика`.
4. Нажмите `Загрузить распакованное расширение`.
5. Выберите папку `.output/cnc1-admin-extension/chrome-mv3-dev`.

#### Firefox

1. Запустите `pnpm run dev:firefox`.
2. В Firefox откройте `about:debugging#/runtime/this-firefox`.
3. Нажмите `Загрузить временное дополнение`.
4. Выберите файл `manifest.json` из папки `.output/cnc1-admin-extension/firefox-mv3-dev`.

- После этого при изменениях в коде WXT пересобирает расширение в `.output/cnc1-admin-extension/<browser>-mv3-dev`.
- Обновляйте расширение вручную на странице расширений или используйте кнопку `Обновить`.
- Если нужна авторизация на `https://cnc1.ru`, откройте сайт в том же профиле браузера до запуска.

### Для Firefox

```bash
pnpm run dev:firefox
```

## Сборка

Для финальной сборки используйте:

```bash
pnpm run build
```

Для создания ZIP-файла:

```bash
pnpm run zip
```

## Полезно

- `wxt` управляет манифестом и сборкой расширения
- `tailwindcss` используется для стилей в интерфейсе
- `@preact/preset-vite` подключает Preact через Vite
